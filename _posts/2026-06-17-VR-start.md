## 1. XR 플러그인 매니지먼트 및 OpenXR 설정 (XR Configuration)

VR 기기와 유니티 에디터를 호환시키는 핵심 플랫폼 설정 단계.

1) XR 플러그인 설치
* `Edit > Project Settings`를 열고 메뉴 하단의 **XR Plug-in Management**를 선택합니다.
* 설치되어 있지 않다면 **Install XR Plugin Management** 버튼을 누릅니다.

2) OpenXR 및 플랫폼 설정
* **PC VR 테스트(또는 단독 구동 빌드)** 탭에서 **OpenXR** 체크박스를 활성화합니다.
* **OpenXR 세팅 패널(`Edit` 버튼)**을 엽니다.
* **컨트롤러 프로필 추가**: `Interaction Profiles` 항목의 `+` 버튼을 눌러 내 기기에 맞는 프로필(예: **Oculus Touch Controller Profile**)을 추가합니다.
* **기능 그룹 활성화**: 메타 퀘스트 기기를 타겟으로 한다면 **Meta Quest Support** 체크박스를 활성화합니다.

## 2. 파일 복제 
* **부분 가져오기(커스텀)**: 샘플에서 필요한 액션 맵만 내 파일로 복사-붙여넣기하여 사용하면 구조가 단순해져 디버깅이 10배 빨라짐. 액션 이름(Select, Move 등)은 원본 규격을 유지하는 것이 안전함.
* **동일 프로젝트 내 복사**: 유니티 에디터 Project 창에서 `Ctrl + D`(Duplicate) 기능을 사용하는 것이 가장 깔끔함.
* **다른 프로젝트로 파일 이동 (핵심)**: 
    * 윈도우 탐색기에서 고유 주민등록번호 역할을 하는 `.meta` 파일은 버리고, 오직 **알맹이 파일(`.inputactions`) 딱 하나만 복사**함.
    * 내 프로젝트 유니티 에디터 창으로 드래그 앤 드롭하면, 유니티 엔진이 내 환경에 맞는 깨끗한 새 `.meta` 파일을 자동 생성하여 GUID 충돌 오류를 방지함.
* **이동 후 필수 작업**: 가져온 새 파일은 반드시 `InputActionManager` 컴포넌트의 `Action Assets` 리스트에 등록하여 전원을 켜주어야 함.

## 3. XR Ray Interactor 주요 옵션 핵심 요약
* **Interaction Manager**: 레이저가 쏜 신호를 받아 처리하는 총괄 매니저와 연결되는 곳임.
* **Interaction Layer Mask**: 레이저가 상호작용할 오브젝트의 레이어를 필터링함 (기본값 Everything).
* **Handedness**: 해당 인터랙터가 어느 쪽 손(Left/Right)인지 결정함.
* **UI Interaction**: 가상 공간의 유니티 UI(Canvas 버튼 등)를 레이저로 클릭하려면 반드시 체크해야 함.
* **Force Grab**: 멀리 있는 물체를 조준하고 그립을 누르면 손으로 휙 날아와 붙게 만드는 기능(포스 그랩)임.
* **Manipulate Attach Transform**: 물체를 잡은 상태에서 조이스틱을 이용해 물체의 위치나 각도를 조작할 수 있게 허용함.
* **Translate Speed**: 물체를 잡고 조이스틱을 앞/뒤로 밀 때 멀어지거나 가까워지는 속도임 (0이면 거리 변화 없음).
* **Rotate Mode / Speed**: 조이스틱 좌/우 조작 시 물체가 회전하는 방식과 초당 회전 각도(`°`) 속도를 지정함.
* **Attach Transform**: 물체가 손에 날아와 붙을 때의 정확한 기준점 좌표(오프셋)를 설정함.
* **Ray Origin Transform**: 레이저 광선이 뿜어져 나오는 시작점 위치를 지정함 (보통 컨트롤러 팁 좌표).
* **Disable Visuals When Blocked**: 레이저가 장애물에 막혔을 때 선(Line Renderer)을 자동으로 숨겨 시각적 어색함을 방지함.


## 트러블 슈팅 : 
* vr 기기와 유니티 프로젝트의 연결이 안되어 동료에게 도움을 구함 
* 기종은 occulus quest2 같은 모델을 사용중인 동료와 대조하여 현재 연결이 제대로 된 상태가 아님을 확인함(배경이 유니티 화면이 아닌 로비 화면) 
* 에어 링크(wifi 로 연결) 해제 후 케이블로 연결하여 제대로 작동됨을 확인
* 착용 중 지속하여 화면이 어두워지고 연결이 끊기는 현상 발견 USB 3.0 을 권장하는 것을 보니 현재 USB와 호환이 제대로 안되어 발생하는 문제로 예상중
